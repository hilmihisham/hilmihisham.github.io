Platform Simulation

A software system that can be used to simulate "Kinetic Art". User can build and configure the platform for starter, and will be able to program each node in the platform to make their own "Kinetic Art" design. Also included is the functionality to save and load their already-made design, which the user can then simulate their finished design straight away without having to create a new one each time.
